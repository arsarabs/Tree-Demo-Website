THE JUNK MOOSE — FULL WEBSITE STRUCTURE
=======================================

OVERVIEW
--------
Single-page marketing website for "The Junk Moose," a junk removal company serving Portland, OR and the surrounding metro area. Built with Next.js 14 (App Router), React 18, TypeScript, Tailwind CSS, and Framer Motion for animations. It's a scroll-driven, section-based landing page designed to convert visitors into leads.

TECH STACK
----------
- Framework: Next.js 14.2 (App Router, static export capable)
- Language: TypeScript
- Styling: Tailwind CSS 3.4 + CSS custom properties
- Animations: Framer Motion 12
- Fonts: Cabinet Grotesk (headings, bold/extrabold) + Switzer (body, medium/regular) via Fontshare
- Utility: clsx + tailwind-merge for conditional class merging
- Images: Next.js Image component (optimized, lazy-loaded)

FILE STRUCTURE
--------------
src/
  app/
    layout.tsx        — Root HTML layout, metadata, SEO, Schema.org JSON-LD, font imports
    page.tsx          — Main page: all sections composed as React components
    globals.css       — CSS reset, Tailwind directives, CSS custom properties
    favicon.ico       — Site icon
    fonts/            — GeistVF.woff, GeistMonoVF.woff (local fallback fonts)
  components/ui/
    marquee.tsx       — Infinite horizontal/vertical scrolling marquee component
    number-ticker.tsx — Animated number counter (springs into view on scroll)
    section-wrapper.tsx — Scroll-triggered fade-in wrapper (Framer Motion)
    spotlight-card.tsx  — Card with mouse-following radial gradient spotlight effect
  lib/
    utils.ts          — cn() helper: merges clsx + tailwind-merge

Config files:
  tailwind.config.ts  — Custom fonts, marquee keyframe animations, content paths
  postcss.config.mjs  — PostCSS with Tailwind plugin
  next.config.mjs     — Next.js config
  tsconfig.json       — TypeScript config
  package.json        — Dependencies and scripts

DESIGN SYSTEM / BRAND
---------------------
Colors (defined as CSS custom properties in globals.css):
  --bg-primary:      #FFFFFF (white backgrounds)
  --bg-section-alt:  #F5F5F5 (light gray alternate sections)
  --bg-section-gold: #92700C (gold/amber accent sections)
  --brand-blue:      #3B82F6 (primary brand blue — CTAs, links, accents)
  --brand-gold:      #92700C (secondary accent — stats section, quote marks)
  --text-primary:    #1A1A1A (near-black headings/body)
  --text-muted:      #6B7280 (gray secondary text)
  --footer-bg:       #1A1A1A (dark footer)

Typography:
  - Headings: font-cabinet (Cabinet Grotesk), extrabold/bold, sizes up to 96px
  - Body: font-switzer (Switzer), regular/medium, typically 16-22px
  - Both loaded from Fontshare CDN

General patterns:
  - Rounded-full buttons (pill shape)
  - rounded-2xl cards with subtle border and shadow-sm
  - Uppercase tracking-wide label text above section headings
  - Smooth scroll between sections (scroll-behavior: smooth)
  - Every section fades in on scroll via SectionWrapper

REUSABLE UI COMPONENTS
-----------------------

Marquee (marquee.tsx):
  - Infinite horizontal or vertical scrolling container
  - Props: reverse, pauseOnHover, vertical, repeat (default 4), className
  - Uses CSS animation defined in tailwind.config.ts (translateX keyframes)
  - Duplicates children N times for seamless loop

NumberTicker (number-ticker.tsx):
  - Animated counter that springs from 0 to target value
  - Props: value, direction (up/down), delay, decimalPlaces, className
  - Uses Framer Motion useSpring (damping: 60, stiffness: 100)
  - Triggers once when scrolled into view (useInView)
  - Updates DOM directly via ref for performance

SectionWrapper (section-wrapper.tsx):
  - Wraps any section with scroll-triggered fade-in animation
  - Props: children, className, id
  - Framer Motion: opacity 0->1, y 20->0, triggers once, -100px viewport margin
  - 0.6s duration, easeOut

SpotlightCard (spotlight-card.tsx):
  - Card with mouse-tracking radial gradient spotlight effect
  - Props: children, className, spotlightColor (default blue at 15% opacity)
  - Tracks mouse position relative to card, shows/hides gradient on hover
  - 600px radial gradient follows cursor

RESPONSIVE BREAKPOINTS
-----------------------
Follows Tailwind defaults:
  - sm: 640px
  - md: 768px
  - lg: 1024px
Mobile-first: base styles are mobile, scaling up at breakpoints.
Key responsive changes:
  - Hero headline: 5xl -> 7xl -> 96px
  - Service grid: 1 col -> 2 col at md
  - Before/after grid: 1 col -> 2 col at lg
  - Owner section: stacked -> side-by-side at lg
  - Stats: stacked -> row at sm
  - Crew strip height: 320px -> 420px -> 500px

INTERACTION PATTERNS
